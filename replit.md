# Sanborns Wishlist - San Valentin

## Overview
Interactive HTML5 banner (300x600px) for DV360/CM360 - Valentine's Day Wishlist with AI integration and email sharing functionality.

## Project Architecture
- **Type**: Static frontend website
- **Stack**: HTML5, CSS3, JavaScript
- **Entry Point**: `index.html`

## Key Files
- `index.html` - Main HTML structure
- `styles.css` - Styling
- `main.js` - Main application logic
- `voice.js` - Voice recognition functionality
- `products-data.js` - Product catalog data
- `gemini-api.js` - AI integration with Google Gemini
- `dynamic-products.js` - Dynamic product handling

## Running Locally
The project uses a static file server on port 5000:
```bash
npm run dev
```

## Deployment
Configured for static hosting - serves all files from root directory.
